Descrição do padrão

O padrão tem a intenção de definir um esqueleto de um algoritmo em uma operação, diferindo em alguns passos de suas subclasses clientes. O padrão deixa subclasses redefinirem certos passos de um algoritmo sem ter que mudar a estrutura do algoritmo.

A classe base declara algorimos de base que serão implementados pelas classes derivadas.

Como é possível notar, o método template chama os outros três seguintes métodos, como se fossem passos. Porém a classe base deixa para as subclasses definirem tais passos.
